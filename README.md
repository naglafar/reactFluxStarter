# React Fluxxor Starter

Simple boiler plate to get you going quickly with es6, react, fluxxor and bootstrap.

    run:
        npm run start
    build:
        npm run build
    lint:
        npm run lint
    test:
        npm run test
    clean:
        npm run clean
    publish to github pages:
        npm run publish

 webpack for code blobbing.

phantom -> karma -> jasmine for tests.
